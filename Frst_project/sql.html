SQL

  • Structured query language (SQL) is a programming language for storing and processing information in a relational database. A relational database stores information in tabular form, with rows and columns representing different data attributes and the various relationships between the data values.

Basic SQL Concepts

    1. SQL (Structured Query Language)
    2. Database management systems (DBMS)
    3. Tables, rows, columns
    4. Primary keys, foreign keys
    5. Data types (INT, VARCHAR, DATE, etc.)

SQL Commands

    1. SELECT (retrieve data)
    2. INSERT (add new data)
    3. UPDATE (modify existing data)
    4. DELETE (delete data)
    5. CREATE (create tables, indexes)
    6. DROP (delete tables, indexes)
    7. ALTER (modify table structure)

SQL Functions

    1. Aggregate functions (SUM, AVG, MAX, MIN)
    2. String functions (CONCAT, UPPER, LOWER)
    3. Date and time functions (NOW, DATE_FORMAT)
    4. Mathematical functions (ABS, ROUND, CEIL)

Common SQL Queries

    1. Retrieve all data: SELECT * FROM table;
    2. Filter data: SELECT * FROM table WHERE condition;
    3. Sort data: SELECT * FROM table ORDER BY column;
    4. Join tables: SELECT * FROM table1 INNER JOIN table2 ON condition;
