C

  • C is a general purposes programing language that is used to develop system software, application software & embedded systtem . It is known as the mother laguages

Basic Syntax

    1. Preprocessor directives (#include, #define)
    2. Variables (declare with type name;)
    3. Data types (int, char, float, double)
    4. Operators (+, -, *, /, =, ==)
    5. Control structures (if, else, for, while, do-while)
    6. Functions (return-type function-name(parameters))

Data Types

    1. Integers (int)
    2. Characters (char)
    3. Floating-point numbers (float, double)
    4. Arrays (type name[size];)
    5. Pointers (type *name;)
    6. Structures (struct name { members };)

Operators

    1. Arithmetic operators (+, -, *, /, %)
    2. Comparison operators (==, !=, >, <, >= , <=) 3. Logical operators (&&, ||, !) 4. Assignment operators (=, +=, -=, *=, /=, %=) 5. Bitwise operators (&, |, ^, ~, <<, >>)

Control Structures

    1. Conditional statements (if, else) 2. Loops (for, while, do-while) 3. Switch statements (switch, case, break) 4. Jump statements (break, continue, return, goto)

Functions

    1. Function declarations (return-type function-name(parameters);) 2. Function definitions (return-type function-name(parameters) { body }) 3. Function calls (function-name(arguments);) 4. Function pointers (type (*name)(parameters);)
