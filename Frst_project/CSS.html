CSS

  • CSS is the Standard for Cascading Style Sheet for designing webpage or web applications
  • CSS describes how HTML elements are to be displayed on screen,paper or in other media

Selectors

    1. Basic Selectors: *, #id, .class, tag
    2. Combinators: >, +, ~
    3. Attribute Selectors: [attr], [attr="value"]
    4. Pseudo-Classes: :hover, :active, :focus
    5. Pseudo-Elements: ::before, ::after

Box Model

    1.box-sizing: content-box, border-box
    2. margin, padding, border
    3. width, height
    4. max-width, max-height, min-width, min-height

Typography

    1. font-family, font-size
    2. font-weight, font-style
    3. text-align, text-decoration
    4. line-height

Colors and Backgrounds

    1. Color Models: RGB, HSL, HEX
    2. background-color, background-image
    3. gradient: linear-gradient, radial-gradient
    4. box-shadow, text-shadow

Layout

    1. Display: block, inline, inline-block
    2. Positioning: static, relative, absolute,
    fixed 3. Float: float, clear
    4. Flexbox: display: flex, flex-direction

Other Features

    1. Border Radius: border-radius
    2. Box Shadow: box-shadow
    3. Text Overflow: text-overflow
    4. Word Wrap: word-wrap
